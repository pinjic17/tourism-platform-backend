## Tourism Platform Backend

This project represents my contribution as a backend developer in a 14-member Agile team working on a tourism platform.  
I contributed to the development of a modular monolith backend built with C# (.NET 8), organizing the system into well-defined domain modules.

The development process was client-driven and iterative, with user interviews conducted every three weeks to gather feedback and refine application features.
The project followed a Scrum-based workflow with development organized into sprints, regular team communication and code reviews.

My work included developing and testing backend functionalities, including unit, integration, query, and command tests, as well as fixing bugs and improving existing features.  
I contributed to the implementation of gamification features for tourists, such as challenges, club systems, XP points and user ranking, improving user engagement within the platform.

I applied Clean Architecture and Domain-Driven Design (DDD) principles across API, Core, and Infrastructure layers.
I used PostgreSQL and Entity Framework Core to manage application data, including creating the database structure, setting up relationships, and applying migrations.
